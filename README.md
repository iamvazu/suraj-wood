## PROJECT OVERVIEW

**Brand Name:** Suraj Wood Products Pvt. Ltd.  
**Industry:** Premium Acrylic Panel Manufacturing for Interior Design  
**Target Audience:** Architects, Interior Designers, Modular Kitchen Manufacturers, Homeowners (Premium Segment)  
**Geography:** India (Primary: Delhi NCR, Bangalore, Mumbai; Secondary: Hyderabad, Chennai, Pune)  
**Brand Positioning:** "The Open Standard in Premium Acrylic Panels" — Transparent, AI-Optimized, Accessible  
**Competitor to Beat:** Praveedh Decor (Mumbai-based market leader)

**Design Philosophy:**  
- **Mobile-First:** 70% traffic from mobile devices  
- **Speed-First:** Core Web Vitals <2s load time  
- **AI-First:** Designed for LLM citation and voice search  
- **Transparency-First:** No gated content, open specifications
